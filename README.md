# Hannah Legal Portfolio

This project is a personal portfolio website for Hannah, an early-career lawyer focused on real estate and commercial law.

## What This Portfolio Highlights

- Professional profile and focus areas
- Article-style legal insights
- Experience timeline and legal background
- Core services in property and commercial work
- Clear contact pathways for consultations

## Built With

- Next.js
- TypeScript
- Tailwind CSS

## Project Goal

To present Hannah's legal work in a way that is polished and credible while remaining realistic about early-career experience and growth.
